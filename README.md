# Gladiator: LLM 기반 전술 전략 RPG

**Gladiator**는 Unity로 개발된 LLM(Large Language Model) 연동형 전술 전략 RPG 프로젝트입니다. 플레이어는 검투사를 관리하고, 고도화된 AI 시스템과 협력하여 전장에서 승리해야 합니다.

# Installation

## Unity 설치
1. Unity Hub를 다운로드하여 설치합니다: [Unity Hub 다운로드](https://unity.com/download).
2. Unity Hub에서 **Unity Editor 6000.0.71f1**을 설치합니다.

## Unity ML-Agents 설치
1. [공식 설치 가이드](https://docs.unity3d.com/Packages/com.unity.ml-agents@4.0/manual/Installation.html#install-ml-agents-package-installation)에 따라 Conda 환경을 설치합니다.
